More Information

Thesis project with 5 team members to develop windows application for managing library instead of using Excel.
Current Features of WU Library Management System:
- Manage Student including add new student, edit student, and delete student
- Manage Book including add new book, edit book, and remove book
- Manage Book Loan and Return Book
- Search Functionality
- Print Functionality
- Reports Generator
- Logs User Activities with TimeStamp
- Show Windows Notification Popup when there are any book loan overdue
- Import from Excel and Export to Excel
- Backup Database and Restore Database Functionality
- Various Settings in Application
- Login History
- Auto Update Application in Background without needing admin privilege
- Host Application Update in GitHub
- Deployment with Squirrel
